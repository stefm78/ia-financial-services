# Chapitre 7 : Agents IA comme Utilisateurs

Chaque agent IA entre dans la banque comme un employé. Il présente ses papiers. Son rôle. Ses limites.

Pas de passe-partout magique. L'agent fraud a droit de lire les transactions, pas de les toucher. L'agent risque peut consulter les portefeuilles, pas les modifier. Zéro confiance, même pour les machines.

Logs complets. Audits. Revues. Comme pour un trader humain. L'IA devient employé virtuel, soumis aux mêmes règles.

**Permissions granulaires. Traçabilité absolue. Responsabilité partagée.**

JPMorgan l'applique déjà. Chaque modèle a son identité, ses droits, ses traces.

L'IA n'est plus un dieu omniscient. Elle est un collaborateur. Contrôlé. Utile. Responsable.

**Dans la banque du futur, même les algorithmes montrent patte blanche.**
